General
Q: What is WebTV?
A: WebTV is an internet access set-top box created by WebTV Networks, Incorporated (often shortened to "WNI"). It initially launched in 1996, was rebranded to MSN TV in 2001, and service officially ended in 2013.
Despite the name, WebTV is not a video streaming platform of any kind, nor does it have any video streaming capabilities. It does have very limited video file support, however.


Q: What is WebTV Viewer?
A: WebTV Viewer (the program you're using right now) is a WebTV simulator (not emulator) initially created by WebTV Networks as a basic web development tool, rather than a fully-featured port of the browser software.


WebTV Viewer
Q: Can you add (x feature) to WebTV Viewer?
A: No. We did not create WebTV Viewer and do not have the source code or any other means to add features to it. It was initially created in 1999 as a basic web development tool - it was never meant to be a full-blown WebTV browser port.


Q: Why can't I make the browser window larger?
A: The Viewer's browser window is a fixed size: 560x420. You can technically bump that up to 640x480 by enabling PAL mode, but this isn't recommended and will break page layouts:
WebTV pages simply were not designed for anything larger than the usual 560x420. In fact, the options bar (one of the basic, built-in components of WebTV) breaks when used in PAL mode!


Q: Where is the fullscreen feature?
A: Unfortunately, there isn't any fullscreen feature. The closest you can get is using Windows Magnifier, which you can find by typing "magnifier" into the Start menu search.


Q: Do I have to use the virtual remote or on-screen keyboard?
A: Nope - Direct mouse clicks and keyboard inputs should work just fine (although you have to press enter on text fields to get them to focus).


Q: Why are the colors so weird?
A: Due to its age, WebTV Viewer was only designed to use 16-bit color, meaning color gradients aren't completely smooth. WebTV Viewer also has some visual glitches that are resolved by scrolling the page or highlighting the affected element with the selection box.


Q: How do I set up MSN Messenger in WebTV Viewer?
A: It's a bit complicated, so using an external MSN Messenger client is recommended. That being said, if you still want to use MSN Messenger in WebTV Viewer, follow these steps:
  1. Open the "messengerProxy" folder located in the Viewer's main folder.
  2. Double-click the "start.bat" file. If you get a warning, ignore it.
If this is your first time using Messenger, make sure to do this as well:
  1. On the menu bar, click Network, then click Set Socks Proxy.
  2. Change the Host Name to either localhost or 127.0.0.1 - either will work fine, since both point to your local computer.


Q: How accurate is WebTV Viewer relative to a real WebTV box?
A: WebTV Viewer is a rather inaccurate port of WebTV, but it's the best we have (since there's no emulator). If you want a slightly more accurate experience, you can try using the Japanese Dreamcast version of WebTV in an emulator like Flycast, but people have reported that it's more annoying (and slower) to use.

In no particular order, here's a list of WebTV Viewer inaccuracies and limitations:
  • € There are little to no animations. The ones that do exist (like the options bar slide) are either too fast or choppy.
  • € Audioscopes (WebTV-exclusive audio visualizers) are very choppy (although this was intentional - the PCs of 1999 didn't have much in terms of video processing power).
  • € Viewer is a lot faster than a real box because it utilizes your current internet connection, rather than the dial-up system it was originally built for.
  • € As noted before, colors are inaccurate because Viewer is limited to 16-bit color, whereas real WebTV boxes can do 32-bit color (equivelant to modern PCs) just fine.
  • € Some TV-related functionality is broken or unavailable (like capturing an image to attach to an email or newsgroup post).


WebTV Servers
Q: What is WebTV Redialed?
A: WebTV Redialed is a revival service for WebTV boxes run by JarHead and Nitrate92. It is compatible with all WebTV and MSN TV units (but not MSN TV 2), as well as WebTV Viewer. If you're reading this FAQ, chances are you got this copy of WebTV Viewer from the Redialed website.


Q: What is HackTV?
A: HackTV is a customized WebTV server run by MattMan69. It has games, weather, news, and more! The server is mainly themed after MSN TV, but you can customize the homepage to reflect other eras of WebTV, from its intial launch to its last days.

HackTV is also the name of a custom WebTV Plus firmware that includes lots of cool stuff that actually makes offline boxes usable, like an HTML previewer, a collection of WebTV songs, and games like Doom, You Don't Know Jack, and even an old copy of MAME.

For more info, see MattMan's website.


Q: What is ZefieTV?
A: ZefieTV is a stock installation of the open-source minisrv software, with a few extras. It's intended to be a demo of what a stock WebTV server is like. If you like what you see and want to host your own, check out minisrv on GitHub.


Q: What is localhost?
A: This represents a server running on your local machine (if you set one up). This option doesn't connect to anything if you don't have a server running.
If you're interested in running your own WebTV server, and don't mind how complicated the setup is, check out Zefie's minisrv software on GitHub.


Q: Why do I get stuck on the "Connecting to WebTV" page or get an error when connecting to a server?
A: After hitting Connect, when you see the "Connecting to WebTV" screen with the city road, type "client:activ" into the URL bar and hit enter.



No, that's not a typo - it's shorthand for "client:activatenetwork", although some people have reported that "client:active" works as well in some cases.


Websites
Q: Why doesn't (modern site) work on WebTV?
A: Keep in mind that WebTV's browser is basically equivelant to Internet Explorer 4. Don't expect modern websites to load or function properly. Yes, this includes YouTube too.


Q: Why doesn't YouTube work on WebTV?
A: WebTV does not support web video of any kind*, including YouTube. It came out long before YouTube was a thing, so it obviously doesn't have support for it.
If you want the technical details, WebTV doesn't support the <video> tag that YouTube (and other video streaming sites) require. WebTV also does not support MP4, webm, or any other modern video format.

*WebTV technically supports MPEG videos, but they have to be under 400KB, meaning they have to be extremely compressed to even play properly.


Q: What kind of video and audio formats does WebTV support?
A: For video formats, WebTV can only play compressed MPEG-1 video (.mpg) and WebTV's proprietary VideoFlash format. The former needs to be compressed to under 400 KB in order to play back, while the latter hasn't been reverse-engineered yet.

For audio formats, WebTV can play MIDI, MOD, XM, MP2, MP3, and RealAudio files. Some later versions may be able to play back WMV and WMA files, but it seems to be a little picky as to what it allows.

For best performance, it is recommended to use files with low bitrates as to make sure it can be played back correctly on real WebTV boxes.

Other
Q: How can I set the Connection Setup as the startup page so I don't have to type the IP in every time?
A: Follow these instructions:
  1. On the menu bar, click on Edit, then click Preferences.
  2. Under Web Mode Home Page, make sure Other is selected.
  3. Set the URL to exactly "file://Disk/Browser/cSetup.htm" (without the quotes). Make sure to erase the "http://" bit first if that's in there.


Q: How do I open the options panel?
A: Press the Del (not Backspace) key on your keyboard. If you don't have one, use the virtual remote or click Browser > Options on the menu bar.
